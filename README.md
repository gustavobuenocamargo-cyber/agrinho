# Projeto: AgroForte Sustentável 🌾🚜

Projeto de página web desenvolvido para a aula de programação da **Professora Regiane**. O objetivo do site é apresentar a força do agronegócio aliada à sustentabilidade, demonstrando o equilíbrio essencial entre a produção de alimentos e a preservação do meio ambiente.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as três tecnologias pilares do desenvolvimento web front-end, de forma totalmente separada e organizada:

* **HTML5:** Para a estruturação semântica do conteúdo da página.
* **CSS3:** Para a estilização, definição da paleta de cores (tons da natureza) e responsividade do layout.
* **JavaScript:** Para a lógica de interatividade e manipulação dinâmica do DOM.

## 📋 Funcionalidades do Projeto

1.  **Layout Responsivo:** O site se adapta a diferentes tamanhos de tela para garantir uma boa experiência de leitura tanto no computador quanto em dispositivos móveis.
2.  **Navegação Suave (Smooth Scroll):** Ao clicar no botão principal, a página desliza suavemente até a seção interativa.
3.  **Simulador de Práticas Sustentáveis:** Uma ferramenta em JavaScript que permite ao usuário selecionar uma técnica agrícola (como plantio direto ou rotação de culturas) e ver instantaneamente na tela os impactos ecológicos gerados por ela.
